# Placar de Wrestling

Um placar digital para competições de Wrestling nos estilos Livre e Greco-Romana.

## Funcionalidades

- **Cronômetro Editável**: Configure o tempo desejado para cada período
  - Presets rápidos: 1:00, 2:00, 3:00 minutos
  - Timer personalizado: defina minutos e segundos manualmente
  - Controles: Iniciar, Pausar, Resetar
- **Placar para Dois Lutadores**: Vermelho e Azul
  - Pontuações: +1, +2, +4, +5 pontos (padrão UWW)
  - Possibilidade de remover pontos (-1)
- **Controle de Período**: 1 a 3 períodos
- **Seleção de Estilo**: Estilo Livre ou Greco-Romana
- **Nomes dos Atletas**: Campos editáveis para identificação
- **Reiniciar Partida**: Reseta todos os valores para uma nova luta

## Como Usar

1. Abra o arquivo `index.html` em qualquer navegador moderno
2. Configure o tempo desejado usando os botões de preset ou o timer personalizado
3. Digite os nomes dos atletas (opcional)
4. Clique em "Iniciar" para começar o cronômetro
5. Use os botões de pontuação para adicionar/remover pontos
6. Use "Reiniciar Partida" para começar uma nova luta
